# Installation

Öncelikle projeyi clonelayın.

```s
git clone https://github.com/lingardium/eduMate
```

# Usage

Projeyi cloneladıktan sonra Visual Studio Code programında açınız.

Linux İçin:

```
cd eduMate
code .
ionic serve
```
